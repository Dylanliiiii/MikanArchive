# Changelog

## Unreleased

### Added

- 首页新增角色空间首屏、透明线稿人物背景、头像状态 hover 展开、滚动遮罩显现、资源轨道、跑马灯和访问量胶囊等动效模块。
- 文章详情页标题元信息区新增浏览量状态和分享入口；浏览量只接真实评论统计服务，未配置时显示“统计不可用”，分享支持原生分享或复制链接。
- 文章详情页右侧目录新增内层小框和当前阅读范围高亮，滚动时会动态圈出当前标题组；移动端继续使用浮动目录入口。
- 新增公开日历设计规格，确认 `/calendar/` 后续作为“我的”下拉中的公开日历页，采用内容文件维护日程并支持年 / 月 / 周 / 日视图。
- 新增 `/calendar/` 公开日历页，支持内容文件维护的公开日程、公开节日、农历日标签、年 / 月 / 周 / 日视图和周 / 日 24 小时时间轴展示。
- 新增 `/site/` 站点概览页，集中展示站点统计、运行时长、最后活动、站点信息、构建信息和公开入口。
- 新增 `/guestbook/` 留言页，未配置评论系统时显示明确占位说明，并保留未来评论服务接入位置。
- 新增“联系我”一级导航入口，下拉包含友链、留言和 QQ 群入口；公开示例仅保留待配置文案，不写真实私人联系方式。
- 新增非主页专用 `ContentGridLayout`，保留导航、页脚、浮动控件和文章宽屏目录，同时移除通用横幅、双侧栏和导航下方通用分类快捷栏。
- 工具导航新增可访问分类标签、滑动指示器、分类数量、分组线、工具图标、域名底栏和响应式三/二/一列卡片。
- 资源模型新增可选 `icon` 字段，并为公开工具示例补充四个分类、九个入口和已安装图标集校验测试。
- 新增工具导航与摘录收藏独立页面，分别承载长期站点入口和面向具体问题的网页线索。
- 新增收藏与归档数据 helper 测试、收藏页面路由契约测试。
- 引入 Firefly 风格主题骨架，新增壁纸横幅、玻璃导航、左右侧栏、文章列表、文章详情、友链墙、搜索和显示设置等主题能力。
- 新增 `LICENSE`，保留 MikanArchive、Firefly 与 fuwari 的 MIT 版权声明。
- 新增 Firefly 重构设计规格，确定下一阶段以 CuteLeaf/Firefly 为底座或强参考重建博客壳层和页面体验。
- 新增 GitHub Pages Actions 工作流，支持通过 Astro 构建并部署 `dist/`。
- 初始化 Astro + MDX + React + Tailwind CSS 的静态站点框架。
- 新增首页、文库、文章详情、收藏、联系我和我的页面。
- 新增搜索弹窗、友链申请弹窗、日历、写作热力图、文章卡片、资源卡片和友链卡片。
- 新增 `content.example/` 示例内容和 SVG 示例资产，支持公开仓库独立运行。
- 新增内容同步脚本和内容校验脚本，支持公开示例内容、本地私有内容目录和远程私有内容仓库。
- 初始化 MikanArchive 项目设计文档、协作规则、项目专属 Skill、开发日志和更新日志。
- 确定第一版采用公开框架仓库与私有内容仓库分离的维护方式。
- 确定第一版导航为 `主页 / 文库 / 收藏 / 联系我 / 我的`。

### Changed

- 首页顶部导航外壳改为浅色半透明长胶囊样式，保留原有站点头像图标、导航图标、搜索和主题按钮，并仅在首页作用域生效。
- 首页从旧 Firefly 横幅双侧栏卡片布局改为专用角色空间布局，继续保留真实顶部导航、页脚、搜索、主题切换和站内链接。
- 导航栏站点图标改用首页头像资产；首页内容入口改为读取真实文章、资源、友链、更新记录和站点配置，不伪造访问量数字。
- 显示设置面板精简为只保留主题色相控制，移除不再适合当前聚焦布局的壁纸模式、水波纹、渐变、樱花特效和文章布局开关。
- 聚焦布局页面会忽略浏览器本地残留的旧壁纸模式，避免文库、收藏、日历等页面被全屏壁纸或透明壁纸状态影响。
- 项目视觉规则补充为默认启用浏览器视觉伴随和视觉对比，覆盖方案讨论、低保真 mockup、布局方向对比和视觉验收。
- “我的”下拉调整为“个人介绍 / 站点概览”，GitHub、B 站和 RSS 移到页脚与站点概览页；页脚新增已运行天数和最后更新时间。
- 站点统计去除重复的“标签”项，只保留使用文章标签口径统计的“分类标签”。
- 删除独立足迹功能页和顶部“足迹”入口；首页原足迹小组件改为个人介绍预览，“我的”下拉中的 `/about/` 入口显示为“个人介绍”。
- 文章总列表页改为无封面长条卡片，每条文章整条可点击进入详情，减少封面字段对列表阅读和点击区域的影响。
- 文库下拉移除独立“标签”入口，保留“文章 / 归档 / 分类标签”；文章页标题从“文库”改为“文章”，并在标题下按分类标签筛选、分组展示文章。
- 分类页更名为“分类标签”，改为基于文章标签统计的扇形分布图和篇数清单；首页侧栏分类统计同步使用文章标签口径，旧 `/tags/` 页面删除。
- 文章归档页活动区改为左侧 GitHub 贡献热力图、右侧周度文章分布热力图，并将文章周度色阶改为固定阈值，6 篇及以上才进入最高色阶。
- 非主页新增“主页 › 当前功能页”面包屑；联系我、收藏等无实质总览页的下拉项直接从主页指向友链、留言、工具导航和摘录收藏等功能页。
- 收藏一级入口不再提供收藏总览功能页；导航下拉仅保留工具导航和摘录收藏，旧 `/resources/` 路径兼容跳转到 `/resources/tools/`。
- 文章归档页新增文章活动热力图、URL 标签筛选、年/月/文章连续时间线，以及总结果、年份和月份三级文章数量；既有分类与未分类归档链接继续兼容。
- 友邻一级入口改为“联系我”，友链内容移动到联系我页面中的友链区域，并新增留言与 QQ 群入口卡片。
- 分类、标签、RSS、个人介绍、搜索、友链和留言等主要非主页可见大标题统一为工具导航同款英文眉标、图标、主标题和说明节奏。
- 全站移除导航下方通用分类快捷栏，避免“主页 / 归档 / 分类 / 更多”在其他页面重复出现。
- 工具导航与摘录收藏统一为白色 / 半透明 section 打底；摘录收藏的标题、英文眉标、图标和描述节奏对齐工具导航。
- 早期仅收藏三页移除顶部通用分类条；当前已扩展为全站移除导航下方通用分类快捷栏。
- 摘录收藏改为整张卡片可点击跳转原文，分类标签、分组标题和标签视觉向工具导航统一，并新增暖色卡片底色、轻边框和 hover 彩色阴影反馈。

- 首页继续使用原 Firefly 横幅与双侧栏；文库、收藏、联系我、个人介绍、文章详情和辅助页面统一改用聚焦内容布局。
- 摘录收藏移除旧大块标题卡，改为与工具导航一致的 section 标题系统并直接衔接内容。
- 资源内容模型新增必填 `kind` 字段，并支持摘录来源、适用场景和收藏日期。
- 将站点 `src/` 主题层整体改造为 Firefly 风格结构，并保留 MikanArchive 的内容同步和公开/私有内容分离边界。
- `npm run build` 现在会自动执行内容同步、内容校验、Astro 构建和 Pagefind 索引生成。
- 文章示例封面字段改为 Firefly 使用的 `image`。
- 为避免无可用修复的 `@swup/astro` high audit 链，当前暂不启用 Swup 无刷新页面过渡。
- 当前视觉方向调整为接近 Firefly / fqzlr.com 的公开博客体验，同时保留 MikanArchive 的内容分离和知识收藏定位。
- 为站内链接和公开资产路径增加 base path 处理，兼容 GitHub Pages `/MikanArchive/` 子路径部署。
- 升级到 Astro 7 依赖栈，改用 `src/content.config.ts` + Content Layer `glob()` loader。
- 移除已弃用的 `@astrojs/tailwind` 集成，改为 Tailwind CSS v3 + PostCSS + Autoprefixer。
- 明确 Node.js 最低要求为 `>=22.12.0`，并新增 `.node-version` 记录推荐版本。
- 将公开文档中的本机绝对路径示例改为相对路径或泛化描述。
- 将内容同步目标统一为文章/profile Markdown 同步到 `src/content/`，JSON 数据同步到 `src/data/content/`，公开资源同步到 `public/assets/`。

### Fixed

- 修复文章详情页右侧目录在长文滚动时随父级离开视口、无法持续固定显示的问题。
- 修复公开日历点击其他日期后右侧“日期详情”不更新的问题，并同步 URL hash 与日期选中态。
- 修复公开日历和聚焦面包屑在暗色模式下视图选中态、日期选中态和当前页文字对比不足的问题。

- 修复非主页固定导航未预留高度、导致顶部分类条在首屏被导航遮挡的问题。
- 修复无侧栏页面仍被通用脚本强制添加侧栏网格、导致正文缩窄的问题。
- 修复工具示例引用不存在的 Iconify 图标导致开发页出现 Astro 错误遮罩的问题。
- 修复 Firefly 组件迁移到 Astro 7 后的模板编译问题，并移除未启用可选页面中的编译阻塞点。
- 修复 GitHub Pages 默认 Jekyll 构建 Astro 源码导致的部署失败问题。
- 通过升级 Astro 依赖链清理 `npm audit --omit=dev` 报告中的安全公告。
